==========
gtasks
==========
CLI interface of `Google Tasks`


install
==========

::
    
    git clone git://github.com/planset/gtasks.git
    cd gtasks
    python setup.py install

create config file

::

    vi ~/.gtasks.conf

.gtasks.conf
::

    CLIENT_ID = ""
    CLIENT_SECRET = ""

CLIENT_ID and CLIENT_SECRET are created in `Google APIs Console <https://code.google.com/apis/console/>`_
